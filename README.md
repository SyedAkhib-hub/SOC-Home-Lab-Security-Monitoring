# SOC Home Lab - Security Monitoring & Log Analysis

## Overview
This project simulates a real-world Security Operations Center (SOC) environment for monitoring and analyzing security logs from Windows and Linux endpoints. The lab focuses on L1 SOC analyst operations such as alert triage, log correlation, and incident investigation.

## Tools Used
- **Wazuh** – Open-source SIEM platform
- **Sysmon** – Windows system monitoring tool
- **Windows & Linux** – Endpoint operating systems

## Objectives
- Build a SOC home lab to simulate a real-world environment
- Configure SIEM to collect and analyze endpoint security logs
- Monitor logs for failed logins, brute-force attempts, and suspicious process executions
- Create dashboards and alerts for potential security incidents
- Practice L1 SOC analyst operations including alert triage and basic incident investigation

## Implementation
1. **Setup Wazuh** for log collection and monitoring
2. **Configure Sysmon** on Windows endpoints to capture detailed system events
3. **Monitor Linux logs** for authentication failures and suspicious activities
4. **Develop dashboards & alerts** in Wazuh to identify potential security threats
5. **Perform analysis** on logs to detect security incidents and respond

## Key Learnings
- Understanding of SIEM architecture and log analysis
- Practical experience in SOC operations (alert triage, log correlation, and incident investigation)
- Skills in monitoring Windows and Linux endpoints
